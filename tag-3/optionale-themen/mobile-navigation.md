# Mobile Navigation

Erstelle eine alternative Navigation für mobile Geräte. Sie soll:

* Einen "Hamburger" rechts oben haben anstelle der Desktop-Navigation.
* Die Menüpunkte sollen versteckt sein.
* Bei Klick auf den Hamburger werden die Menüpunkte eingeblendet.
* Nochmaliger Klick schliesst das Menü wieder.

Rahmenbedingungen:

* Die Navigation soll ohne JavaScript funktionieren.
* Du kannst im Internet Beispiele nutzen.
* Manche Beispiele nutzen eine versteckte Checkbox.&#x20;

Zusatzaufgaben:

* Blende das Menü dynamisch ein (slide).
* Verwandle den Hamburger beim Öffnen des Menüs in ein Kreuz.
* Füge ein Untermenü hinzu. Nutze dazu neben den Menü-Texten einen Pfeil nach unten (Menü öffnen) bzw. nach oben (Menü schliessen).
