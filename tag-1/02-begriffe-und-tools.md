# Begriffe und Tools

## Begriffe

Einige Begriffe sollten auseinander gehlaten werden:

* **Internet:** Ein Netz von dezentral vernetzten Geräten, über welche viele Dienste laufen wie E-Mail, Video-Calls, Messenger, WWW, Streaming, Internet-Telefonie, Filetransfer etc.
* **Web / WWW:** Weltweit mit Hyperlinks vernetzte HTML-Dokumente, die übers Internet-Protokoll HTTP/HTTPs übertragen werden.&#x20;
* **Website**: Gesamtheit aller Seiten, welche zu einem Webauftritt gehören
* **Webseite**: Eine einzelne Seite innerhalb eines Webauftrittes
* **Homepage**: Startseite einer Website (Home)

## Tools

**Texteditor für HTML & CSS:** Im Prinzip wird nur ein Texteditor benötigt, z.B. [Notepad++](https://notepad-plus-plus.org/downloads/). \
Hilfreicher ist aber eine Entwicklungsumgebung, z.B. [Visual Studio Code](https://code.visualstudio.com/download).&#x20;

**Bildbearbeitung:** Eine Software wie [Photoshop](https://www.adobe.com/ch_de/products/photoshop.html), [Gimp](https://www.gimp.org/downloads/) oder [IrfanView](https://www.irfanview.com/main_what_is_ger.htm) oder ein Online-Dienst wie [Online Image Editor](https://www.online-image-editor.com/) oder [Bulk Resize Photos](https://bulkresizephotos.com/).&#x20;

**Kompatibilität:** [caniuse.com](https://caniuse.com/) sagt dir, von welchen Browsern ein Feature unterstützt wird und wie verbreitet diese Browser aktuell sind.

**Validierung:** [HTML-Validator](https://validator.w3.org/) und [CSS-Validator](https://jigsaw.w3.org/css-validator/)

**Testen und Fehlersuche:** Dein Browser bietet dir umfangreiche Developer Tools. Nutze im Browser die Taste `F12` oder Rechts-Klick & "Untersuchen" auf ein Element einer Website.&#x20;
